<h1 align="center">PinkieTech</h1>

<p align="center"><strong>現場の力を、技術で引き出す。</strong></p>

PinkieTechは、実際に仕事が行われる「現場」から出発するエンジニアリングチームです。
人と仕事を理解し、AI・IoT・OSSを手段として使いながら、実践できる改善を一緒につくります。

## 私たちが大切にすること

- **人を第一に** — 技術を目的にせず、人を支えるために使います。
- **現場から始める** — 物理的な場所に限らず、仕事が行われる状況と声を理解します。
- **小さく試し、育てる** — 完璧な構想を待たず、使える一歩から改善を重ねます。
- **知識をひらく** — OSSを尊重し、再利用できる形で知識を残します。
- **話しかけやすく** — 難しい言葉に頼らず、誠実で相談しやすい関係をつくります。
